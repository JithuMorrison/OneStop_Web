# Product Overview

SSN Connect Portal (OneStop) is a comprehensive college portal for SSN College of Engineering that consolidates multiple student and faculty services into a unified platform.

## Core Features

- **Authentication**: Role-based access (students, teachers, admin) with email validation
- **Social Networking**: Posts, announcements, likes, comments, following system
- **Academic Tools**: CGPA calculator, timetable management, materials sharing
- **Event Management**: Announcements with registration, exam schedules, event tracking
- **OD Claims**: Student on-duty claim submission and teacher approval workflow
- **Messaging**: Real-time chat between users with auto-expiring messages (10 hours)
- **File Management**: Upload and share academic materials with categorization
- **Admin Dashboard**: User management, teacher account creation, system oversight
- **Clubs**: Club management with moderators and member tracking
- **Gamification**: Badges, achievements, streaks, and credit system

## User Roles

- **Students**: Access to all features, can create posts/announcements, submit OD claims
- **Teachers**: Can approve OD claims, create exam schedules, manage content
- **Admin**: Full system access, user management, club creation, query responses

## Technology Approach

Hybrid architecture with both MongoDB (legacy backend) and Supabase (new implementation) for gradual migration.
